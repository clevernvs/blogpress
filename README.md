## Utilizando o projeto

1. Instale as dependências do projeto
~~~~
npm install
~~~~

2. Compile e inicie o servidor de desenvolvimento
~~~~
npm run serve
~~~~

3. Acessar a aplicação pelo navegador: http://localhost/
